# 2.2.1-Inlamningsuppgift
Skapa ett formulär där användaren kan skicka in sitt namn och ålder. Skriv sedan ut en hälsning som inkluderar användarens namn och en meddelande som är beroende av användarens ålder. Om användaren är under 18 år, skriv "Hej [användarens namn]! Du är för ung för att använda den här webbplatsen.
